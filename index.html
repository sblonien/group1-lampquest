Game Description

The goal of the game is to terraform each planet that you are assigned. Each planet comes up with some initial resources and energy. For each assigned planet, you will be given a list of items and their required quantity that must be present in the planet to make it habitable.

You need to use the automated robots from the factory to create new resources that are required for supporting life in the planet. Each robot will cost you some initial energy.

There are two types of robots: combiners and diffusors. The combiners can combine two or more available resources to produce new items. These combiners requires some energy in each production. The diffusors can create multiple items from a single available resource. These diffusors will release some energy in each production. However, diffusor come with certain energy limit, that is once the energy limit of the diffusor is reached, it won't be able to produce any more items and will remain idle.

If a robot cannot produce item(s) due to insufficient resources or energy, the robot will be automatically disabled. You can only enable them when the requirement to produce item(s) is fulfilled.
